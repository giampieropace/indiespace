# Welcome to the  Indie 🌌 Space Website

IndieSpace.net is a personal project aimed at making the IndieWeb more accessible to people.

Making a personal website is no easy task if you're not familiar with coding, and even if we have now a lot of platforms allowing one to make a websites using a graphical user interface with drag & drop blocks, they are not cheap nor fully customizable.

I want to be immediately clear, so that the hardcore and purist supporters of the IndieWeb don't freak out: the main goals  of this project are:

1. To provide practical guides on how to create a website.
2. To promote the principles of the IndieWeb (at least those that may be most accessible to non-experts)
3. (& 4) Are works in progress.
